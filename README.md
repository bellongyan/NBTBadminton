# NBTBadminton

基于 HarmonyOS 的校内羽毛球论坛系统

通过使用 ArkTS 和 ArkUI 构建 HarmonyOS 原生应用，设计并实现基于 HarmonyOS 类似于“中羽在线”的校内羽毛球论坛系统

前端使用 ArkTS 和 ArkUI 进行编写，后端使用 SSM 进行开发，项目前后端分离
